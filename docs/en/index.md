## Welcome to OpenHEXA

OpenHEXA is an open-source data integration platform developed by [Bluesquare](https://bluesquarehub.com) to facilitate data integration and analysis workflows, particularly in the context of public health projects.

![Workspace Switcher](../assets/images/openhexa.png){ width=70% align=center }


This documentation provides comprehensive guides for using OpenHEXA effectively, from basic navigation to advanced pipeline development.
