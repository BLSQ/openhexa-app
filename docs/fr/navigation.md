Ce guide couvre les principales fonctionnalités de navigation qui vous aident à vous déplacer entre les organisations et les espaces de travail, et à trouver rapidement des ressources sur la plateforme.

## Navigation des organisations et des espaces de travail

### Comprendre la hiérarchie

OpenHEXA organise le contenu dans une structure hiérarchique :

- **Organizations** : Entités de niveau supérieur contenant plusieurs espaces de travail
- **Espaces de travail** : Environnements collaboratifs où vous travaillez sur des projets, des jeux de données et des pipelines spécifiques
- **Ressources** : Éléments individuels comme les jeux de données, les pipelines, les fichiers et les connexions au sein des espaces de travail

Vous pouvez appartenir à plusieurs organisations et avoir différents niveaux de permissions dans chacune. Par exemple, vous pourriez être :

- Admin dans une organisation mais Viewer dans une autre
- Avoir accès à tous les espaces de travail de l'Organisation A mais seulement à des espaces de travail spécifiques de l'Organisation B (voir l'exemple ci-dessous)
![Organization-Workspace Relations](../assets/images/navigation/orgs_ws_relation.png){ width="60%" }
- Être Editor dans certains espaces de travail mais Viewer dans d'autres


### Vue au niveau de l'organisation

Si vous êtes membre de l'organisation sans permissions Admin ou Owner, la vue au niveau de l'organisation fournit un aperçu de tous les espaces de travail et jeux de données auxquels vous pouvez accéder au sein de votre organisation actuelle.

![Organization View](../assets/images/navigation/org_view.png)


#### Accéder à la vue

1. Dans le panneau de navigation de gauche, cliquez sur le nom de l'organisation.
2. Vous verrez une grille ou une liste de tous les espaces de travail et jeux de données accessibles.

![Switch organization](../assets/images/navigation/switch_org.png){ width="30%" }

## Recherche Spotlight

La recherche Spotlight est une fonctionnalité puissante qui vous aide à trouver rapidement tout ce à quoi vous avez accès dans OpenHEXA, dans tous vos espaces de travail. Cliquez sur la barre de recherche en haut de la barre latérale gauche.

Le panneau de recherche apparaît centré sur votre écran.

![Organization-Workspace Relations](../assets/images/navigation/spotlight.png)

### Utiliser la recherche Spotlight

1. **Commencez à taper** : Entrez quelques lettres de ce que vous cherchez.
2. **Voir les résultats** : OpenHEXA affiche instantanément les résultats correspondants regroupés par catégorie dans tous les espaces de travail sélectionnés.
3. **Accéder aux résultats** : Cliquez sur n'importe quel résultat pour accéder directement à cette ressource.

!!! info "Conseils de recherche"
	- **Correspondance partielle** : Vous n'avez pas besoin de taper le nom complet, les correspondances partielles fonctionnent bien
	- **Filtrage par catégorie** : Les résultats sont automatiquement regroupés par type pour une navigation facile
	- **Accès rapide** : Appuyez sur `Cmd + K` (sur macOS) ou `Ctrl + K` (sur Windows/Linux) pour un accès plus rapide 

## Changement de contexte d'espace de travail

Lorsque vous avez accès à plusieurs espaces de travail, vous pouvez facilement basculer entre eux en utilisant le menu de sélection d'espace de travail dans le coin supérieur gauche de l'écran.

### Changer d'espace de travail

1. **Trouvez le sélecteur d'espace de travail** : Cherchez le menu déroulant dans le coin supérieur gauche.
2. **Cliquez pour développer** : Cliquez sur le nom de l'espace de travail actuel pour voir tous les espaces de travail disponibles.
3. **Sélectionnez la destination** : Choisissez l'espace de travail vers lequel vous souhaitez basculer.
4. **Navigation automatique** : Vous accéderez directement à l'espace de travail sélectionné.

En savoir plus sur les espaces de travail dans la section [Espaces de travail](workspaces.md) du manuel d'utilisation.


## Changer d'organisation

Lorsque vous appartenez à plusieurs organisations dans OpenHEXA, vous pouvez facilement basculer entre elles pour accéder à différents ensembles d'espaces de travail.

### Comment changer d'organisation

1. **Trouvez votre organisation actuelle** : Cherchez le nom de votre organisation active en haut de la barre latérale gauche.
2. **Accédez à la vue de l'organisation** : Cliquez sur le nom de l'organisation pour voir la [vue au niveau de l'organisation](#vue-au-niveau-de-lorganisation).
3. **Changez d'organisation** : Cliquez à nouveau sur le nom de l'organisation pour obtenir une liste de toutes les organisations auxquelles vous appartenez.
4. **Sélectionnez la destination** : Choisissez l'organisation vers laquelle vous souhaitez basculer.
